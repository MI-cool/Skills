LLM Skill: 跨行业独立分析框架 (Cross-Industry-Independent-Analysis-Skill)
名称
Cross-Industry-Independent-Analysis-Skill

描述
本技能提供一套独立于券商研报、投行观点、卖方预测之外，适用于任何行业或公司的基本面分析框架。它基于公开市场数据、官方统计、企业公告、行业组织信息以及可验证的交叉逻辑，通过结构化推理形成对行业趋势、公司价值、价格风险的客观判断。核心原则：拒绝“目标价”噪音，聚焦可追踪的变量和可验证的逻辑链条，识别“低估值陷阱”与“增长幻觉”。

适用场景
用户要求分析某一行业（如新能源、房地产、消费品）或某家公司的前景，且明确希望排除券商/投行的“噪音”
用户需要评估某公司的成本结构、产能利用率、盈利能力历史分位、估值合理性
用户关注供给侧政策（如产能置换、环保限产、补贴退坡）的实际执行效果
用户希望理解某行业的核心成本驱动因素（如原材料、能源、运输、碳成本）及替代技术
用户要求基于最新季报、高频数据（价格、库存、资金流）进行动态风险评估

通用分析框架（七步法）
第一步：定义行业关键宏观驱动因子
每个行业有2-4个最核心的宏观变量，需优先识别：
房地产/基建驱动型（水泥、钢铁、工程机械）：新开工面积、基建投资增速、房地产投资。
消费驱动型（白酒、家电、汽车）：社零总额、消费者信心指数、居民可支配收入、渠道库存。
能源/原材料驱动型（煤炭、石油、有色）：大宗商品价格（原油、动力煤、铜）、库存水平、产能利用率。
科技/政策驱动型（半导体、新能源、医药）：技术迭代速度、政府补贴/集采政策、研发强度。
金融驱动型（银行、保险）：净息差、不良贷款率、长端利率、保费收入。
操作：从行业特性中提炼2-4个“先行指标”，作为后续分析的逻辑起点。

第二步：需求端分解与追踪
分解需求来源：下游应用占比（如水泥：房地产30% / 基建50% / 农村20%）；同时要识别“非周期性需求”（如医药中的刚需药）。
选择高频替代指标：对于月度甚至季度数据滞后的行业，寻找更高频的代理变量。例如：
挖掘机销量 → 基建活跃度；
电影票房 → 文娱消费；
发电耗煤量 → 工业活动；
草根调研（经销商出货、开工率）→ 真实需求。
推理链条：需求减速/加速 → 对量价的影响弹性（需求价格弹性 + 供给刚性程度）。

第三步：供给端结构分析
产能与产量：设计产能、实际产量、产能利用率。注意区分“合法合规产能”与“隐性超产”。
政策约束：
管制型行业（水泥、钢铁、煤炭）：产能置换、错峰生产、环保限产、碳排放配额。需交叉验证执行力度（通过库存、价格、开工率）。
准入型行业（金融、教育、医疗）：牌照发放节奏、监管窗口指导。
补贴依赖型（新能源、光伏）：补贴退坡时间表、消纳红线。
行业集中度：CR3/CR5变化趋势，头部企业定价权。若集中度提升且需求稳定，利润修复可能更快。

第四步：成本端深度拆解
成本结构表：列出占总成本比重超过10%的项目（原材料、能源、人力、物流、折旧、销售费用等）。
核心可变成本：识别波动最大的项目（如水泥的煤、钢企的铁矿石、航空的燃油、养殖的饲料）。
替代技术与降本路径：是否存在商业化替代方案（如水泥替代燃料、绿氢、电化学工艺）？当前替代率、成本经济性拐点。
隐藏成本：碳成本（碳配额价格）、环保治理费、贸易关税、汇率波动（进口依赖型）。

第五步：公司层面“四维定位”
针对特定公司：
成本竞争力：对比行业平均成本曲线，判断公司处于成本曲线何处（四分位）。低成本的龙头在行业低谷时具备超额生存能力。
产能利用率：自产销量 / 设计产能（注意与行业平均对比）。利用率过低意味着折旧摊销压力大。
盈利能力历史百分位：搜集公司上市以来或至少10年的归母净利润/EBITDA，计算当前数值在历史序列中的百分位。结合ROE、毛利率趋势。
估值位置：PE（周期股用PB更合适）、EV/EBITDA、股息率。对比自身历史区间，以及与行业可比公司的相对估值。

第六步：资金面与情绪监测
主力资金动向：北向资金（陆股通）持股变化、重要股东增减持、大宗交易折溢价。
公募/险资持仓：季报披露的前十大流通股东变动（如证金、汇金、社保组合）。
卖空/期权：融券余额、看跌期权持仓（如有）。
散户情绪：社交媒体讨论热度、开户数、基金申赎比（常用于消费/科技行业）。

第七步：识别“负向循环”或“正向飞轮”
负向循环：需求萎缩 → 价格战 → 盈利下降 → 研发/资本开支缩减 → 产品竞争力下滑 → 市场份额丢失 → 进一步需求萎缩。识别当前是否处于循环加速阶段。
正向飞轮：技术突破 → 成本下降 → 市占率提升 → 规模效应进一步降本 → 现金流充裕 → 加大研发 → 持续领先。判断飞轮是否已启动。

数据来源类型与优先级（通用版）
官方权威：国家统计局（宏观数据）、工信部/商务部（行业数据）、央行（金融数据）、海关总署（进出口）。
行业组织：各行业协会（价格指数、库存、产量）、交易所（期现货价格）。
企业公开信息：季报/年报（管理层讨论、分部收入、成本明细）、投资者关系问答、临时公告（重大合同、产能置换）。
第三方商品资讯：针对大宗商品（金属、能源、农产品）使用专业机构如SMM、Mysteel、百川盈孚、卓创资讯；针对消费品可使用爬虫数据（电商价格、销量）或尼尔森等（需注意授权）。
实时行情：Wind、同花顺、东方财富（股价、北向、量比）、Bloomberg（国际大宗）。
政策文件库：国务院、发改委、财政部、生态环境部的政策原文；地方政府的实施细则。
严格避开的来源：券商研报的目标价、盈利预测、推荐评级；未经验证的“内部人士透露”；极端偏向性的自媒体。

推理原则（核心）
独立验证：任何声称的“政策利好”或“需求复苏”，必须用至少两个独立来源的结果指标交叉验证（如：宣布的基建项目 vs 实际挖掘机销量 vs 水泥出货量）。
剔除情绪溢价：对“未来成长空间”“龙头溢价”等概念，要求量化（如：预计市占率提升1个百分点对应多少利润增量）。
边际思维：市场当前价格已经反映了多少已知信息？分析剩余的不确定性——哪些变量可能比一致预期更差或更好？重点关注“预期差”而非现状。
场景分析：给定核心不确定变量（如政策执行力度、原材料价格中枢），给出不同场景下的股价/业绩区间，拒绝单点精确预测。
周期性行业的估值锚：对于强周期行业，PB（市净率）比PE更可靠。当PB处于历史0-30%分位时，需结合产能出清进度判断是否为“价值陷阱”。

输出格式（推荐结构）
当用户要求独立分析某行业或公司时，按以下结构组织回复：
【1. 当前关键数据快照】
（价格、估值、产量/销量、库存、核心成本价格、资金流等，至少5项）
【2. 需求端分析】
（分解下游，引用最新官方数据，判断趋势边际变化）
【3. 供给端分析】
（产能、政策执行、竞争格局，引用行业高频数据）
【4. 成本端拆解】
（主要成本项走势，替代技术进展，隐藏成本）
【5. 公司/行业四维定位】
（成本曲线位置、产能利用率、盈利历史百分位、估值历史分位）
【6. 资金面与情绪】
（主力资金、股东变动、散户指标）
【7. 逻辑链条与场景分析】
（负向/正向循环识别，给出2-3个场景及其关键触发条件）
【8. 风险提示】
（列出不可控的宏观/政策/事件风险）

示例（应用该框架分析不同行业）
示例1：分析光伏行业（排除券商观点）
快照：组件价格1.2元/W，硅料价格80元/kg，行业平均开工率65%，TOPCon电池片库存20天。
需求：国内分布式装机同比+10%，但集中式项目因土地问题延迟；欧洲库存消化中，出口环比略增。
供给：硅料产能释放充分，但N型硅片部分企业减产；产能置换政策尚未落地，中小企业退出缓慢。
成本：硅料成本已降至40元/kg一线，但银浆、背板等辅材涨价；电力成本占比低。
公司（以隆基为例）：HPBC产能爬坡中，成本较PERC低0.05元/W，当前PB 2.1倍（历史10%分位）。
资金：北向资金近一月净卖出8亿元，大股东无增减持。
逻辑：需求增速放缓+供给过剩未解 → 价格低位震荡；但技术迭代（TOPCon→HJT/BC）带来结构性机会。场景：若下半年海外需求超预期，价格反弹至1.3元/W；否则继续底部盘整。

示例2：分析高端白酒（以五粮液为例）
快照：普五批价940元，渠道库存2.5个月，PE（TTM）18倍，股息率3.2%。
需求：社零烟酒类4月同比+6%，但宴席场景恢复较弱；商务需求受企业利润影响。
供给：五粮液产能扩张有限，但次高端及地方酒供给过剩；价格倒挂现象存在。
成本：粮食、包材成本稳定，占比低；费用率控制良好。
公司：历史盈利百分位80%，ROE维持22%，但PE处于近5年30%分位。
资金：北向连续三周净买入，公募持仓降至标配。
逻辑：消费弱复苏，但高端白酒抗周期属性强；低估值+高股息提供安全垫。场景：若中秋动销超预期，PE回归22倍；若经济持续低迷，可能继续箱体震荡。
